# LightGBM

저희 callback 기능을 사용하여 LightGBM의 성능을 단 한 줄의 코드로 시각화 할 수 있습니다.

```python
from wandb.lightgbm import wandb_callback
import lightgbm as lgb

....

gbm = lgb.train(params,
                lgb_train,
                num_boost_round=20,
                valid_sets=lgb_eval,
                valid_names=('validation'),
                callbacks=[wandb_callback()])
```
