# XGBoost

**저희의 callback 기능을 사용해서 다른 버전의 XGBoost 모델 간 결과를 비교하실 수 있습니다.**

```python
bst = xgb.train(param, xg_train, num_round, watchlist,
                callbacks=[wandb.xgboost.wandb_callback()])
```
