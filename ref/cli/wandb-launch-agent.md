# wandb launch agent

**Usage**

`wandb launch-agent [OPTIONS] PROJECT`

**Summary**

Run a W\&B launch agent (Experimental)

**Options**

|              |                                                       |
| ------------ | ----------------------------------------------------- |
| **Option**   | **Description**                                       |
| -e, --entity | The entity to use. Defaults to current logged-in user |
| -q, --queues | The queue names to poll                               |
| --help       | Show this message and exit.                           |
