# wandb pull

**Usage**

`wandb pull [OPTIONS] RUN`

**Summary**

Pull files from Weights & Biases

**Options**

| **Options** | **Description** |
| :--- | :--- |
| -p, --project | The project you want to download. |
| -e, --entity | The entity to scope the listing to. |
| --help | Show this message and exit. |
