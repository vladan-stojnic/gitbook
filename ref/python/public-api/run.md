<h3 id="scan_history"><code>scan_history</code></h3>

<a target="_blank" href="https://www.github.com/wandb/client/tree/c129c32964aca6a8509d98a0cc3c9bc46f2d8a4c/wandb/apis/public.py#L1231-L1285">View source</a>

<pre><code>scan_history(
    keys=None, page_size=1000, min_step=None, max_step=None
)</code></pre>

Returns an iterable collection of all history records for a run.


#### Example:

Export all the loss values for an example run

```python
run = api.run("l2k2/examples-numpy-boston/i0wt6xua")
history = run.scan_history(keys=["Loss"])
losses = [row["Loss"] for row in history]
```
